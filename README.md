
# Stocku

O programa Stocku serve para gerenciamento básico de estoque através de um database local contendo o id, nome, quantidade e preço dos produtos. O presente projeto tem como finalidade a conclusão do TDE da disciplina Estrutura de Dados na UniFacema.

## Instruções para compilação

Caso deseje compilar o projeto no Windows, existem dois métodos:

1. Baixe diretamente o [Make para Windows](https://gnuwin32.sourceforge.net/packages/make.htm) e execute via terminal no diretório do projeto
‎ 
```
make
```

2. Usando [Chocolatey](https://chocolatey.org/install). Primeiro instale esse gerenciador de pacotes. Após instalado, abra o CMD ou Powershell com privilégio de administrador e execute o seguinte comando, confirmando a permissão caso necessário
‎ 
```
choco install make
```
Após isso, basta executar via terminal no diretório do projeto o comando
‎ 
```
make
```
